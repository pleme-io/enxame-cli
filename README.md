# enxame-cli

[![crates.io](https://img.shields.io/crates/v/enxame-cli.svg)](https://crates.io/crates/enxame-cli)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE)

enxame — the ENXAME BitTorrent command-line client: inspect .torrent files and magnet links (name, files, info-hash, trackers) and verify content. Part of the pleme-io ENXAME suite.

Part of **ENXAME** — the pleme-io BitTorrent suite: one sans-io engine and
thin faces, with GUIs on every platform (terminal, web, native desktop).
See [`theory/ENXAME.md`](https://github.com/pleme-io/theory/blob/main/ENXAME.md).

## Use

```toml
enxame-cli = "0.1"
```
